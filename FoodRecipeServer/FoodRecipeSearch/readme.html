Additional Comments

Question 1-1

Full-stack app created to facilitate communication between MERN frameworks - 4pts

Question 1-2

Clicking a Populate DB button in the React App sends a POST request to a Node app that adds 782 of the 787 recipes from the recipe.json file to the DB based on a Mongoose model - 2pts.

Populate DB button disappears after it has been clicked and the DB has been populated with recipes from the dataset. - 1pts

After the data has been written to the DB, a message is returned to and displayed in the UI as confirmation - 1pts
    --> Used alert feature

Question 1-3

Clicking an Add Recipe button in the tReact App displays a form hat allows the user to add a new recipe to the database - 3pts.

The user can create a recipe by adding one ingredient at a time using input fields for quantity, measurement, and ingredient name - 3pts.

Question 1-4

The input field for the name of the ingredient uses an auto-suggest feature to suggest ingredients to the user based on ingredients that are already in the database - 4pts.
    --> Used regex to percolate data to datalist on app load after local fetch is done

A table of ingredients is displayed each time the user adds an ingredient -1pt.

Upon addition of a recipe to the database using the Add Recipe Form, a message containing a count of the number of recipes in the database is displayed in the UI - 1pt.
    --> read DB on app load, read DB on add recipe, read DB on populate DB.

CSS, JS used based on best practices. Removed all previous errors and warning - 5pts

HTML page readme.html provided with EC2 working evidence - 5pts
